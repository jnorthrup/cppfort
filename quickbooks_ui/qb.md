# QuickBooks Mode (qb mde)

## Overview
QuickBooks Mode (`qb_mode`) transforms the default UI into a specialized accounting interface mimicking the functionality and aesthetics of QuickBooks Online.

## Configuration
This mode is controlled via `qb_config.json`.
- `qb_mode`: Set to `true` to enable by default.

## Features
- **Account Creation**: Enhanced modal with industry-specific autopopulation.
- **Visuals**: Premium "Green" aesthetic, glassmorphism, and responsive layout.
