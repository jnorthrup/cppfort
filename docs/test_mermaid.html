5. Key Design Principles

  • Semantic Preservation: Analysis happens once on AST, results flow through pipeline
  • No Re-lowering: SON is the final IR stage (no LLVM IR)
  • Direct Codegen: SON → C++ generation for simplicity and speed
  • JIT on FIR: Allocation passes run on high-level FIR before optimization
  • Separation of Concerns: Each stage has clear responsibilities
