# Minimal Dogfood Implementation Plan
 
## Overview (UNBIASED)

Implement the closed-loop **dogfooding** infrastructure for cppfort, enabling it to parse `bootstrap_tags.cpp2` through the pipeline: cpp2 source → canonical AST → (optionally) C++ emission → executable.

**Key distinction**:
- **Dogfooding**: cppfort parses cpp2 source (parser correctness)
- **Self-hosting**: cppfort parses cppfort's own cpp2 source (completeness)
- **Transpilation**: cppfront transpiles cpp2 to C++ (temporary bootstrap tool)

**This plan focuses on dogfooding** - verifying cppfort can parse cpp2 syntax correctly. Self-hosting and transpilation are separate concerns.

## Phase 5: Create Test Script `scripts/test_selfhost.sh`

### Content
```bash
#!/bin/bash
set -e

# Test 1: Baseline with cppfront
echo "=== Test 1: cppfront baseline ==="
cppfront -p -q src/selfhost/bootstrap_tags.cpp2 -o /tmp/tags_cppfront.cpp
clang++ -std=c++20 /tmp/tags_cppfront.cpp -o /tmp/tags_cppfront
BASELINE_OUTPUT=$(/tmp/tags_cppfront)
echo "Baseline output:"
echo "$BASELINE_OUTPUT"

# Test 2: cppfort native
echo ""
echo "=== Test 2: cppfort native ==="
./build/src/selfhost/cppfort src/selfhost/bootstrap_tags.cpp2 -o /tmp/tags_cppfort
NATIVE_OUTPUT=$(/tmp/tags_cppfort)
echo "Native output:"
echo "$NATIVE_OUTPUT"

# Compare
echo ""
echo "=== Comparison ==="
if [ "$BASELINE_OUTPUT" == "$NATIVE_OUTPUT" ]; then
    echo "SUCCESS: Outputs match!"
    exit 0
else
    echo "FAILURE: Output mismatch"
    echo "Diff:"
    diff <(echo "$BASELINE_OUTPUT") <(echo "$NATIVE_OUTPUT")
    exit 1
fi
```
